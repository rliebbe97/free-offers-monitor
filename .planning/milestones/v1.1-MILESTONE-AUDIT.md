## E2E Flow Verification

**Source row → Dashboard display:**

1. `fetchActiveSources` queries `sources` (no type filter) → returns reddit + bump rows
2. `runIngestionCycle` calls `createAdapterForSource(source)` per row
3. Factory dispatches `case 'bump'` → `createTheBumpAdapter(source.identifier)`
4. `TheBumpAdapter.fetchNewPosts(since)` runs pagination loop → returns `RawPost[]`
5. Each post upserted to `posts` table, Tier 0 runs inline
6. Passed posts enqueued to `tier1_queue` via `enqueueTier1`
7. Tier 1/2 pipeline processes as normal (shared across all sources)
8. `runIngestionLoop` drives cycle every `POLL_INTERVAL_MS`
