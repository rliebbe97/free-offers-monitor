<action>
Create the directory `packages/db/src/migrations/` if it does not exist.

Create `packages/db/src/migrations/001_seed_thebump_sources.sql` with the following content:

```sql
-- Migration: 001_seed_thebump_sources
-- Adds TheBump subforum source rows to the sources table (type='bump').
-- Safe to re-run: ON CONFLICT (identifier) DO NOTHING.
--
-- Apply via Supabase SQL Editor or CLI:
--   psql $DATABASE_URL -f packages/db/src/migrations/001_seed_thebump_sources.sql

-- TheBump Freebies subforum
INSERT INTO sources (type, identifier, config)
VALUES (
  'bump',
  'https://community.thebump.com/categories/freebies-and-deals',
  '{
    "base_url": "https://community.thebump.com",
    "subforum_path": "/categories/freebies-and-deals",
    "max_pages": 5
  }'::jsonb
)
ON CONFLICT (identifier) DO NOTHING;
```

Key design decisions:
- `type` = `'bump'` (matches Phase 6 dispatch factory filter)
- `identifier` = full subforum URL (UNIQUE constraint makes re-run safe)
- `config` JSONB contains `base_url`, `subforum_path`, and `max_pages` for adapter use
- `last_polled_at` is omitted (defaults to NULL; first adapter run sets it)
- `ON CONFLICT (identifier) DO NOTHING` makes the migration idempotent
- Only one source row for now (the freebies-and-deals subforum); additional subforums can be added later with separate INSERT statements in the same or new migration file
- Migration naming convention: `{NNN}_{description}.sql` — establishes this as the standard

Note: The ROADMAP success criteria mention "two seeded rows" for both freebies and deals categories. However, the RESEARCH section shows a single combined "freebies-and-deals" category URL. If TheBump has separate subforum URLs for freebies vs deals, add a second INSERT. If they are the same category page, one row suffices. The implementer should check the live site. If two distinct subforums exist, add:

```sql
-- TheBump Deals subforum (if separate from freebies)
INSERT INTO sources (type, identifier, config)
VALUES (
  'bump',
  'https://community.thebump.com/categories/deals',
  '{
    "base_url": "https://community.thebump.com",
    "subforum_path": "/categories/deals",
    "max_pages": 5
  }'::jsonb
)
ON CONFLICT (identifier) DO NOTHING;
```
